About This App

This mini app helps build a clean bibliographic database from OpenAlex, Web of Science, and Scopus sources. Everything runs in the browser, so uploaded files stay on this computer unless you download or share the exported files yourself.

OpenAlex Extraction

Search OpenAlex, preview retrieved records, remove review-like titles when needed, and export CSV, JSON, or Web of Science-style TSV.

Web of Science Merge

Upload up to 20 tab-delimited savedrecs files, combine them into one table, remove duplicates, and download TSV or CSV outputs.

Scopus Conversion

Convert Scopus CSV exports into the same Web of Science-style tab-delimited structure used by the rest of the app.

Cross-Source Merge

Merge converted tab-delimited outputs from multiple sections, deduplicate across sources, preview results, and export a final database.

Light Analysis

Group the final merged database in the browser with TF-IDF, summarize topic groups, inspect top keywords, and export grouped files.

Data Handling

File parsing, conversion, merging, deduplication, and downloads happen locally in browser memory. The only network request is the OpenAlex query when you use the OpenAlex extraction tab.
